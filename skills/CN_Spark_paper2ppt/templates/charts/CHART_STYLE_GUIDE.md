# Chart Style Guide
document explanation(It doesn't affect the process, it only helps with understanding）：本文件在新增、修改或验证 templates/charts 下 SVG 图表时读取；它规定图表颜色、字体、标签、兼容性和学术 deck 的克制表达。

Charts must follow `references/shared-standards.md`. Use theme color for target series, gray for baseline, same-hue variations for related series, green only for positive direction, and brick red only for risk or key warnings. Verify axis scales, units, legends, source notes, values, and PPT-compatible SVG primitives.
